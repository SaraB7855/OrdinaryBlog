Programiranje za početnike

Saveti & Iskustva

Programiranje je moguće naučiti na fakultetu ili kursevima, ali je takođe moguće samostalno naučiti uz internet.
O tome da se i programiranje može samostalno naučiti najbolje svedoči istraživanje Stack Overflowa koje je otkrilo da polovina developera u svetu nema visoko obrazovanje u svojoj struci, a mnogi nikakvo visoko obrazovanje uopšte.


Odakle početi?

Postoji dosta raznih kurseva, ali nasa preporuka je da se krene od HTML, CSS & JavaScript kurseva.

Html je standardni prezentacijski jezik za izradu web-stranica, odnosno jezik u kome se koriste određeni elementi predstavljeni oznakama (tagovima) kako bi se izgradila web-stranica.
CSS predstavlja stilski jezik koji se koristi za dizajn izgleda web-stranice, odnosno koristi se za prikaz sadržaja napisanog u html dokumentu na određeni način. Online postoji mnoštvo kurseva i tutorijala pomoću kojih možete naučiti osnove html i css jezika, takođe, postoji i mnoštvo resursa koje možete koristiti u web dizajnu.
JAVASCRIPT je objektni skriptni jezik, čija je osnovna svrha korišćenja vezana za poboljšanje dinamike određene stranice. Zapravo je stvar u tome da kada se koristi HTML kod, njegova uloga je samo da oblikuje određene elemente stranice, kao što su recimo tekst, tabele, linkovi ili forma, te da ih nakon toga uredi, ali ovaj kod nema ulogu da odredi na koji način će se konkretni elementi ponašati. Upravo tada se u čitav procesu uključuje JavaScript, koji ima značajnu ulogu u kreiranju ponašanja ovih elemenata. A kada se on kombinuje sa HTML - om i CSS - om, tada nastaje takozvani dinamički HTML (DHTML).

Resursi za učenje :

Mozzila Developer Network

Mozilla Developer Network ili MDN je jedan od najpopularnijih besplatnih resursa za učenje web dizajna sa najbogatijim sadržajem koji se redovno ažurira. Osim dokumentacije vezane za html i css, možete učiti i Java Script, SVG, WEB API i drugo.

W3 schools

Poseduju veliku zbirku tutorijala koji se kreću od onih za početnike do onih za napredni nivo. W3Schools uči i JavaScript i još mnogo toga. Jedan je od najboljih resursa za učenje web dizajna, sa detaljno objašnjenim načinima i principima po kojima funkcionišu kodovi.

HTML cheatsheet

HTML Cheat Sheet uključuje potpunu listu html elemenata, opise tih elemenata, praktične primere koda i slično. Svi elementi su organizovani po kategorijama, a možete ih preuzeti i kao PDF.

CSS cheatsheet

CSS Cheat Sheet olakšava rad sa css-om, objašnjava css u potpunosti. Svaka css oznaka je objašnjena do detalja, čemu služi i kako funkcioniše. Ovaj besplatan resurs mogu koristiti i početnici i iskusni dizajneri.

Sjajna stvar u vezi programiranja

Nemate šta da izgubite, a možete da dobijete toliko mnogo. Pored moguće karijere i zarade, učite se jednom posebnom načinu razmišljanja koji će vam vam pomoći u raznim stvarima, čak i svakodnevnom životu, posebno u planiranju, donošenju odluka i rešavanja problema.
Možete početi i sa tutorijalima koji se nalaze na YouTube platformi, sa W3Scools materijalima ili sa Udemy kursevima koji su veoma povoljni. Za srednji ili napredni nivo znanja možete koristiti Coursera kurseve ili kurseve koje nudi Udacity.
